## 🚀 部署到 Vercel

### 方式一：通过 Vercel Dashboard（推荐）

1. **准备 Git 仓库**
   ```bash
   cd C:\Users\admin\Codes\Minner-fun
   git init
   git add minner-fun-docs vercel.json
   git commit -m "Initial commit: Docusaurus documentation site"
   ```

2. **推送到 GitHub**
   - 在 GitHub 创建新仓库
   - 推送代码：
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git branch -M main
   git push -u origin main
   ```

3. **在 Vercel 导入项目**
   - 访问 [https://vercel.com](https://vercel.com)
   - 点击 "Add New Project"
   - 导入 GitHub 仓库
   - Vercel 会自动检测到 `vercel.json` 配置
   - 点击 "Deploy"

### 方式二：使用 Vercel CLI

```bash
# 安装 Vercel CLI
npm i -g vercel

# 在项目根目录运行
cd C:\Users\admin\Codes\Minner-fun
vercel

# 按提示操作，选择配置：
# - Setup and deploy: Yes
# - Scope: 选择你的账户
# - Link to existing project: No
# - Project name: minner-fun
# - Directory: ./
# - Override settings: No
```

## 🔧 本地开发

```bash
# 进入项目目录
cd minner-fun-docs

# 安装依赖
npm install

# 启动开发服务器
npm start

# 访问 http://localhost:3000
```

### 语言切换测试
- 英文：http://localhost:3000

## 📝 添加新文档

### 添加英文文档
在 `minner-fun-docs/docs/` 对应目录创建 `.md` 文件

### 添加中文翻译
在 `minner-fun-docs/i18n/zh-Hans/docusaurus-plugin-content-docs/current/` 对应路径创建同名 `.md` 文件

### 自动生成侧边栏
文档会自动被 `sidebars.ts` 中的 `autogenerated` 配置捕获并显示

## 🎨 自定义

### 修改站点信息
编辑 `minner-fun-docs/docusaurus.config.ts`

### 修改首页
编辑 `minner-fun-docs/src/pages/index.tsx`

### 修改样式
编辑 `minner-fun-docs/src/css/custom.css` 或 `src/pages/index.module.css`
