launchpad平台
项目介绍：代币销售平台，包括项目方注资，平台方配置销售参数，投资方参与注册于购买。
单个代币销售合约有factory合约创建。
factory合约：工厂合约，用于创建salse合约。提供常规的salse创建情况查询
salse合约：平台方设置基本的销售信息，比如注册时间，销售时间，token数量，最大参与人数等
通过线下加签线上验签的方式。
销售完毕，项目方提取销售收益与未销售完的token

allocationStacking:销售前的质押合约，质押平台代币，获取购买额度，然后在注册的时候，会设定此用户的解锁时间，注册之前用户都是可以随时提取的

链下：计算销售额度，抽签，和加签
