## 安装

```bash
# 安装 Rust
rustup toolchain install 1.87.0 // 在当前项目中使用 Rust 1.87.0 版本
rustup override set 1.87.0

sh -c "$(curl -sSfL https://release.anza.xyz/v3.0.13/install)"

echo 'export PATH="/home/minner/.local/share/solana/install/active_release/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc


cargo install --git https://github.com/solana-foundation/anchor avm --force
avm install 0.31.1
avm use 0.31.1
```
rustc --version
solana --version
anchor --version
solana config get       // 查看当前环境配置
solana config set -ul    // 切换到local环境
solana config set -ud   // 切换到dev环境
solana-keygen new       // 生成新的密钥对
solana address         // 显示当前密钥对的公钥地址
solana balance      // 显示当前密钥对的余额
solana airdrop 2       // 领dev的空投
solana-test-validator  // 开启本地测试环境
```
