## 💡 为什么需要测试？

- **安全性**: 智能合约一旦部署就无法修改，测试可以提前发现漏洞
- **可靠性**: 确保合约在各种情况下都能正常工作
- **成本控制**: 避免因bug导致的资金损失
- **用户信任**: 经过充分测试的合约更容易获得用户信任
