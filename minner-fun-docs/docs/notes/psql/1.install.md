


安装 PostgreSQL 服务端和客户端（psql）。
--fix-missing 会尝试跳过无法下载的包或重新获取缺失的依赖，保证安装尽量完成。
```
sudo apt install postgresql postgresql-contrib --fix-missing

psql --version

sudo systemctl status postgresql


```



## 默认账号登录
```
sudo -i -u postgres
psql
```

## 创建数据库，添加用户
```
-- 创建数据库
CREATE DATABASE alpha_trace;

-- 创建用户，并设置密码
CREATE USER alpha_user WITH PASSWORD 'yourpassword';

-- 给用户授权
GRANT ALL PRIVILEGES ON DATABASE alpha_trace TO alpha_user;

-- 退出 psql
\q
```
