# StartLaunch Design
A fair launch period. Before the official Uniswap swap begins, there is a fixed buy-in window for a set period of time. During this window, users can only buy, not sell. The buy price is the initial price.
The swap calculation uses the methods from Uniswap V4's FullMath library.

Token issuance charges a fixed fee, 10^-6 ETH
